aixpert – Apply/Check/Save/Undo/Query security settings¶

  • Synopsis

  • Requirements

  • Parameters

  • Examples

  • Return Values

  • Status

Synopsis¶

This module facilitates a. Application of security rules on the system using aixpert b. Checking of security settings c. Undo of the previously applied set of security rules d. Saving of security rules in normal or abbreviated mode.

Requirements¶

The below requirements are needed on the host that executes this module.

  • AIX

  • Python >= 2.7

  • Should be run as ‘root’ user

Parameters¶

mode (True, str, None)

Specifies the action to be performed mode=apply, applies the security settings based on the level specified or profile provided. If both are provided, ‘level’ will take precedence. mode=check, checks the security settings against the previously applied set of rules or the provided ‘profile’ file mode=save, saves the security settings for the level specified or based on the specified ‘profile’ file. - If abbr_fmt_file, is provided, the security rules are saved in the abbreviated file format. - If norm_fmt_file is provided, the security rules are saved in normal format. mode=undo, undoes the previously applied security settings. mode=query, get the type of the profile applied on the system.

level (optional, str, None)

Specifies the security level settings to be applied or saved. high - Specifies high-level security options low - Specifies low-level security options. medium - Specifies medium-level security options. default - Specifies AIX® standards-level security options. sox-cobit - Specifies SOX-COBIT best practices-level security options.

profile (optional, str, None)

When mode=apply, specifies the profile to be applied on the system

When mode=check, specified the profile to be used to check the security settings.

abbr_fmt_file (optional, str, None)

When mode=save or mode=apply, specifies the file where the security settings needs to be saved in abbreviated format.

norm_fmt_file (optional, str, None)

When mode=apply or mode=save, specifies the file where the settings should be saved in normal format.

Examples¶

- name: "Save default level rules in normal format"
  aixpert:
     mode: save
     level: default
     norm_fmt_file: /home/kavana/norm.xml

- name: "Apply using saved profile"
  aixpert:
     mode: apply
     profile: /home/kavana/norm.xml

- name: "Undo the settings"
  aixpert:
     mode: undo
